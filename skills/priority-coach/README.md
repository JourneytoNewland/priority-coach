# AI优先级教练 (Priority Coach) - 使用指南

> 专为感知型人格(P人)设计的AI优先级教练，效率提升5-10倍

---

## 🎯 这是什么？

**priority-coach** 是一个Claude Code Skill，通过三大核心机制帮你管理任务：

1. **脑部清理协议** - 零散想法 → 结构化清单
2. **目标对齐逻辑** - 基于长期目标进行价值评分（1-5分）
3. **时间保护机制** - 生物钟匹配 + 20%留白原则

**适用人群：** 感知型人格（P人）、容易分心、任务爆炸的人

---

## 🚀 快速开始

### 1. 初始化（首次使用）

```
priority-coach 初始化
```

系统会询问：
- 未来3个月想完成的1-3件事
- 这些事情对你的意义
- 每周愿意投入多少小时

### 2. 日常使用

| 时间 | 命令 | 作用 |
|-----|------|------|
| **早上** | `priority-coach 帮我规划今天` | 生成今日计划 |
| **任务爆炸时** | `priority-coach [你的零散想法]` | 快速整理优先级 |
| **下班前** | `priority-coach 今日总结` | ⭐ 每日复盘 |
| **周末** | `priority-coach 本周总结` | ⭐ 绩效周报 |

## 📊 核心功能

### 1. 智能优先级排序

基于你的长期目标，自动计算任务价值分数（1-5分）：

```
你: 今天要写报告、回邮件、准备会议...

我:
📋 核心项目 (价值5分):
1. 撰写Q1报告 [3番茄钟] - 直接推进目标

📋 高价值任务 (价值4分):
2. 准备会议材料 [2番茄钟]

🧹 琐碎杂事 (价值2分):
- 回复邮件 [建议集中处理]
```

### 2. 时间规划

生成符合生物钟的时间表，预留20%留白：

```
⏰ 今日时间表

【深度工作】9:00-11:30
- Q1报告 [3番茄钟]

【协作时段】14:00-16:00
- 会议准备 [2番茄钟]

【留白时段】17:30-18:00
- 机动时间
```

### 3. 每日复盘 ⭐ 新功能

自动分析当日完成情况：

```
🌙 每日复盘

📊 今日完成度: 75%
🍅 番茄钟: 6/8个
🎯 目标推进: +5%

💡 明日优化:
- "测试环境配置"比预期多花1小时
- 建议: 明日预留更多缓冲时间
```

### 4. 绩效周报 ⭐ 新功能

回顾本周成就：

```
📊 第10周成就

🎯 目标1: 大数据智能体
- 投入: 28小时
- 进度: 35% → 55% (+20%) 🚀

💪 目标3: 跑步
- 本周: 26公里 ✅

🏆 TOP3成就:
1. 完成核心功能开发 (5分)
2. 修复关键bug (5分)
3. 集成测试通过 (4分)
```

## 🔧 技术实现

### 核心脚本

| 脚本 | 功能 |
|-----|------|
| `calculate_value_score.py` | 价值评分算法 |
| `analyze_escape_patterns.py` | 逃避模式分析 |
| `daily_review.py` | ⭐ 每日复盘生成 |
| `generate_weekly_report.py` | ⭐ 绩效周报生成 |

### 价值评分算法

```
Value = (目标权重 × 相关性) × (紧迫性×0.3 + 重要性×0.7)
```

**输出：** 1-5分，用于排序和优先级判断

## ⚠️ 常见问题

**Q: 计划赶不上变化怎么办？**

A: 20%留白时间就是应对变化的。突发任务评估价值分数后决定是否执行。

**Q: 我总是高估/低估时间？**

A: 系统会学习你的时间估算模式，使用2周后会自动校准。

**Q: 长期目标变了？**

A: 使用 `priority-coach 重新设定目标` 随时更新。

**Q: 这个Skill适合J人吗？**

A: 本Skill专为P人设计。J人可能更需要"执行跟踪"工具。
